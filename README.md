# Awesome Browser Testing Tools

A curated list of **browser-based testing tools** that run directly in the web browser.  
These tools are useful for quick checks without installing any software.

---

## Input Device Testing

### Mouse

- **GearTest – Mouse Test**  
  Browser-based mouse testing tool for checking click behavior and detecting double-click issues.  
  https://www.geartest.online/mouse-test/

### Keyboard

- **GearTest – Keyboard Test**  
  Online keyboard testing tool to verify key registration and detect ghosting or key conflicts.  
  https://www.geartest.online/keyboard-test/

---

## Display / Screen Testing

- **GearTest – Screen Test**  
  Browser-based full-screen color tests for detecting dead pixels and display issues.  
  https://www.geartest.online/screen-test/

- EIZO Monitor Test  
  https://www.eizo.be/monitor-test/

---

## Audio Testing

- **GearTest – Audio Test**  
  Online audio testing tool for checking microphone input and speaker output directly in the browser.  
  https://www.geartest.online/audio-test/

- Online Mic Test  
  https://www.onlinemictest.com/

---

## Notes

All tools listed here run in the browser and are intended for **quick diagnostics**, not deep benchmarking.

---

## Scope

This list focuses on tools that run directly in the browser.  
Hardware-specific and device-focused tools are intentionally kept minimal.
